# Unblock clients in UBNT Unifi Controller

## Problem
* Accidentally blocked a client and want to unblock it

## Solution
1. Download latest Unifi Controller
2. Goto -> "Clients" -> "All Configured Clients" -> Select the client -> click "Unblock"
